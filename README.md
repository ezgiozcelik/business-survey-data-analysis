# business-survey-data-analysis
R code for cleaning, merging, analyzing, and visualizing firm-level survey data to examine the relationship between market capacity characteristics, political influence, and political regime type.

*Author*
All code in this repository was written by Ezgi Ozcelik in 2025 as part of her academic research project.

*Project Overview*
This project examines how political regime type influences the relationship between firms' market capacity characteristics and political influence.

*Research Questions*
What are the effects of firms' market capacity characteristics on their influence over governments? 
How does the political regime type of a country in which a firm operates moderate these effects?

*Data Sources*
EBDR's Business Environment and Enterprise Performance Survey (BEEPS) data 
Varieties of Democracy (V-dem) dataset accountability indicators
World Bank Official Exchange Rate dataset

*Methods*
Data importing and cleaning
Merging datasets
Variable construction
Statistical analysis (hierarchical random effects models to analyze multi-level data)
Data visualization (marginal effects plots)

*Note* 
This repository is intended as a code sample demonstrating data management, analysis, and visualization workflows.
